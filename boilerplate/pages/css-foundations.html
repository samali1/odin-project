CSS Foundations

Selectors

Selectors are used to select the HTML elements you want to style.

There are these selector types:

  • Universal
  • Type selector
  • Class selector
  • ID selector
  • Grouping selector
  • Chaining selector

Universal Selector

The universal selector selects all elements in the document. Denoted by: *

Example: 

        * {
            color: red;
        }

Type Selector

The type selector will select all the elements of the given element type. Denoted by type-id

Example: 

        p {
            color: red;
        }

Class Selector

The class selector will select all the elements with the given class attribute. Denoted by .class-id

Example: 

        .red-text {
            color: red;
        }

ID Selector

The ID selector will select the element with the given ID attribute. Denoted by #id

Example: 

        #red-text {
            color: red;
        }

Hasthtag immediately by the case-sensitive value of the ID attribute. Common pitfall is going overboard with the ID usage. Don't need to go crazy, chill out.

Grouping Selector

The grouping selector will select all the elements that are grouped together. Denoted by selector1, selector2, selector3, ...

Example: 

        p, .red-text, #red-text {
            color: red;
        }

Chaining Selector

The chaining selector will select all the elements that are chained together. Denoted by selector1 selector2 selector3 ...

Example: 

        .subsection#preview {
            color: red;
        }

Side notes

Descendent combinators: multiple selectors that can be grouped together. Dependent on a hierarchy.

Example:

        .ancestor .contents {
            color: red;
        }

Properties of CSS

Color and Background-Color

color --> element text color, background-color --> background color of element
Can accept HEX, RGB, HSL values

Example: 

        .red-text {
            color: red;
            color: #ff0000;
            color: rgb(255, 0, 0);
            color: hsl(0, 100%, 50%);
        }

Tyopgraphy and Text-Align

  • font-family: single val or csv. can use "font family name" or generic-family-name
  • font-size: #px
  • font-weight: normal, bold, bolder, lighter, 100-900
  • text-align: center, etc.
  • height, width: auto, #px, ... also good practice to include this with the img attribute because helps with reserving the space before pages load in
