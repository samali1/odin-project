Basics of HTML

HTML, CSS, and Javascript work together to help generate dynamic content. HTML is the structure of the page, CSS is the style, and Javascript is the behavior.

The first line in your HTML file should be the DOCTYPE declaration. This tells the browser what version of HTML the page is written in. The next line is the html tag, which wraps all the content on the page. The head tag contains the title of the page and any other meta information.

Body Element

The body tag contains the content that is generated on the page. Here we can use various tags such as h1 through h6 for headings, p for paragraphs, ul and ol for lists (unordered and ordered, respectively), and a for links.

Sidenote: for headings, h1 is the largest. h6 is the smallest heading. For example, "Basics of HTML" is an h2 tag, and "Body Element" is an h4 tag.

Different text modifications you can do:

  • strong> will bold
  • em will italicize
  • !--- comment --->

Images and Links

The anchor tag a can be used to anchor different elements on a page. For example, if you'd like to redirect to a link, you can use the a tag. The img tag is used to display images on a page. You can specify the source of the image using the src attribute. You can also specify the alt text, width, and height of the image using the alt, width, and height attributes, respectively.

Anchor elements

The a tag allows us to create a link in HTML. The href attribute is used to specify the URL of the page the link goes to. The target attribute is used to specify where the linked document will be opened.

There are four possible values for the target attribute:

  • blank will open the link in a new window
  • self will open the link in the same frame
  • parent will open the link in the parent frame
  • top will open the link in the full body of the window

Another attribute that can be used with the a tag is the rel attribute. This attribute specifies the relationship between the current document and the linked document.

  • noopener prevents the opened link from gaining access to the webpage from which it was opened
  • noreferrer prevents the opened link from knowing which webpage or resource has a link to it

This behavior is added for opening links in new tabs due to security. The prevention access of "noopener" prevents phishing attacks. In good practice, it is smart to always pair a target="_blank with a rel="noopener" to prevent any security issues.

Absolute and Relative Links

Absolute links link to pages on other websites on the internet. They are usually made up of the following parts: protocol://domain/path, e.g. https://www.google.com. Relative links link other pages within our own website.

Image element

The img tag is self-closing. It embeds an image into the document. The src attribute is used to specify the URL of the image. The alt attribute is used to specify the alternative text for the image. The width and height attributes are used to specify the width and height of the image, respectively.

  • The four formats used: jpg, gif, png, svg
  • Always specify the alt, width, and height

Basic tips to keep in mind

  • Always declare the document type
  • Always specify the alt, width, and height for images
  • Always quote attribute values and keep them lower-case
  • Never skpi the title element
  • ALways set the viewport; width and initial-scale
